🔧 DÉPANNAGE RAPIDE

Problème : L'extension ne s'affiche pas
Solution  : Ctrl+F5 pour actualiser la page (pas juste F5!)

Problème : Les couleurs ne changent pas
Solution  : Vider le cache (Ctrl+Shift+Delete)

Problème : La popup ne s'ouvre pas
Solution  : Vérifier que l'icône est épinglée dans la barre d'outils

Problème : Erreurs dans la console
Solution  : Consulter INSTALL.md (dépannage)

Problème : Site ralentit
Solution  : Désactiver "Animations smooth" dans les paramètres
