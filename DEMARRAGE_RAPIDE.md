# ⚡ Démarrage Ultra-Rapide (2 minutes)

## 🎯 En 3 étapes seulement

### Étape 1 : Préparer (1 min)
```
1. Créer un dossier "CrunchyrollModernizer"
2. Copier TOUS les fichiers fournis dedans
3. C'est tout !
```

### Étape 2 : Installer (1 min)
```
1. Ouvrir Google Chrome
2. Taper: chrome://extensions/
3. Activer "Mode de développement" (toggle en haut à droite)
4. Cliquer "Charger l'extension non empaquetée"
5. Sélectionner le dossier "CrunchyrollModernizer"
6. ✅ BOOM ! Extension installée !
```

### Étape 3 : Utiliser (immédiat)
```
1. Aller sur https://www.crunchyroll.com
2. Actualiser la page (Ctrl+F5)
3. 🎨 L'interface devient noire avec accents orange !
4. Cliquer l'icône orange pour les paramètres
```

## ❌ Ça ne marche pas ?

**Première chose : actualiser avec Ctrl+F5** (pas juste F5!)

Sinon :
1. Aller à `chrome://extensions/`
2. Désinstaller l'extension (poubelle)
3. Refaire l'installation (étape 2 ci-dessus)
